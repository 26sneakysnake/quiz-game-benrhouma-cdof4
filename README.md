# Quiz Game

A console-based quiz game where players answer questions to win the game.

## Installation

1. Clone the repository

git clone https://github.com/26sneakysnake/quiz-game-benrhouma-cdof4

2. Navigate to the project directory

cd quiz-game-benrhouma-cdof4

## How to Run

python src/game.py
